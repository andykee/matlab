# ~/dev/matlab

A collection of MATLAB scripts to make life easier and fix several annoyances.

## Installation

1. Place scripts in ~/dev/matlab

2. Add the following line to startup.m:
    addpath(genpath('~/dev/matlab'));

3. Enjoy MATLAB++
